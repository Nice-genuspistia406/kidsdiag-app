## 📊 Using the Dashboard

The dashboard serves as your main hub for all health-related entries. The interface uses a clean design to help you find what you need without delay.

* Adding a New Entry: Click the Add button in the top corner to create a new health record. You can log symptoms, dates, and severity levels.
* Viewing History: The main screen displays a list of past entries. You can filter these by date to see changes over time.
* Exporting Records: Use the File menu to save your logs as a text file. This allows you to print reports before you visit a clinic or doctor.
* Backup Data: The software stores files in your user folder. Copy this folder to an external drive or cloud service to keep your logs safe.

## 🛡️ Privacy and Data Security

Your data stays on your computer. The application does not send information to external servers or remote databases. All logs that you create reside in a local text database stored within your user profile directory. You maintain full control over your information. To delete your data, simply remove the associated folder located in your Documents directory.

## 🛠️ Troubleshooting Common Issues

Errors occur rarely, but use these steps if the application fails to start or behaves unexpectedly:

* The app does not open: Ensure that you are using a user account with administrative privileges. Restart your computer and try opening the program again.
* The installer fails: Check that your antivirus software does not block the installation. Temporarily disable security software if you trust the source, then restart the installation process.
* Display issues: If the windows look distorted, check your Windows display settings. Set your scaling to 100% or 125% to ensure all buttons and menus appear correctly.
* Data errors: If your logs do not save, make sure you have enough free storage space on your primary hard drive. Ensure the application maintains permission to write files to your Documents folder.

## 📝 Best Practices for Logging

Keep your records helpful by following these simple habits:

* Consistency: Use the same terms for symptoms, such as Fever or Cough, to make them easy to filter later.
* Detail: Add specific notes about duration or frequency in the comment box provided in each entry form.
* Timeliness: Enter information shortly after you notice a change in symptoms. This reduces the risk of forgetting minor details.
* Preparation: Print your summary logs the day before any scheduled appointment. This ensures you have the data ready to share.
